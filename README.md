# Strudel Music Maker – README

## Controls

- **Preprocess**: Processes the code in the text editor in preperation for play.  
- **Proc & Play**: Preprocesses the code and immediately starts playback.  
- **Play / Stop**: Plays or stops the currently playing code.  
- **Save / Load**: Save your current settings to a `.musicmachine.json` file or load a previously saved file.  
- **Gain Slider ([ / ])**: Adjust the audio gain. Use the slider or keyboard shortcuts `[` to decrease and `]` to increase.  
- **Song Speed (Numerical Input)**: Change the song speed (default 28). Can also be adjusted with `{` and `}` keys.  
- **Phaser Slider**: Adjust the phaser effect intensity.  
- **Instrument Toggles (ON / MUTE)**: Turn individual instruments on or off.  
- **Toggle Dark Mode (Ctrl + D)**: Switch between dark/light theme.  
- **Play/Pause (Ctrl + Enter)**: Play/Pause.
